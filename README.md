# AI Business Insight Dashboard

An interactive business analytics dashboard built using Python and Streamlit to analyze sales data and generate business insights.

## Features

- Interactive dashboard for business analytics
- Sales analysis by region
- Identification of top selling products
- KPI metrics such as total sales and total orders
- Region based filtering for deeper insights

## Dataset

This project uses the Superstore Sales Dataset which contains information about:

- Customer orders
- Product categories
- Regions and shipping
- Sales revenue

## Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## How to Run

Install required libraries:

pip install streamlit pandas matplotlib seaborn

Run the dashboard:

streamlit run dashboard.py

Open in browser:

http://localhost:8501

## Project Outcome

The dashboard helps visualize business performance and identify trends in sales data, enabling better data-driven decision making.
